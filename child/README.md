## Install

```bash
cd `$01PROJCMD_dir`
cmake -B build
cd build/
make 01PROJCMD
./01PROJCMD --flags # See Usage
```

### Basic Usage

| Flag         | Values | Description                         |
| ------------ | ------ | ----------------------------------- |
| `--flagName` |        | Run 01PROJTEMP in a flags mode      |
| `-v`         |        | output version information and exit |
| `-h`         |        | output help information and exit    |
