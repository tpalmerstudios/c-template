## Install

```bash
cd `$01PROJCMD_dir`
cmake -B build
cd build/
make 01PROJCMD
./01PROJCMD --flags # See Usage
```

| Flag        | Values | Description                         |
| ----------- | ------ | ----------------------------------- |
| `--testing` |        | Run 01PROJTEMP in testing mode      |
| `-v`        |        | output version information and exit |
